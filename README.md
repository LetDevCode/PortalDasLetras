# 📚 Portal das Letras

Projeto desenvolvido em Python utilizando Flask.

## Funcionalidades
- Cadastro e login de usuários
- Controle de sessão
- Dashboard administrativo
- CRUD de livros
- Pesquisa de livros
- Estatísticas e gráficos
- Interface responsiva

## Tecnologias
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Chart.js
- Git/GitHub
